# House Price Predictor

This is my first machine learning project.

I used Linear Regression from scikit-learn to predict house prices based on house area and number of bedrooms.

While making this project, I learned:

- What Linear Regression is
- How fit() trains a model
- How predict() makes predictions
- The difference between machine learning and normal programming

Example:

Input:
Area = 1200 sq ft
Bedrooms = 2

Output:
Predicted Price ≈ ₹66 lakh
